Builder Placing an Order
  1. 1
    Go to the portal
    localhost:3000/express
    Redirects to login automatically. Bookmark it.
  2. 2
    Log in
    Enter username and password provided by ACC.
  3. 3
    Project Details
    Client name, site address, city, delivery date.
  4. 4
    Select Scope
    Check Cabinets, Trim, and/or Doors. Only selected sections appear.
  5. 5
    Cabinets
    Add finish groups — paint/stain/melamine, color, door style. Add rooms, then drop in your cabinet list with sizes.
  6. 6
    Trim
    Pick style and spec level. Enter LF for base, crown, shoe, and openings. System calculates board-feet to order.
  7. 7
    Doors
    Add line items — type, size, species, core, hardware, qty.
  8. 8
    Review & Submit
    Check the summary. Hit Submit Order.
    Confirmation screen shows your job number. ACC gets the PDF immediately.
  9. 9
    View past orders
    Click My Orders in the header anytime.
PM Receiving an Order
  1. 1
    Check the inbox
    Email hits residential@
    advancedcabinets.net     the moment builder submits.
    Subject: New Express Order — ACC-XXXX — Client Name
  2. 2
    Open the PDF
    Full breakdown — client, builder, all specs, door pricing, trim BF. Print or save to job folder.
  3. 3
    Find it in the system
    localhost:3000/jobs
    Express jobs show an orange Express badge. Status starts at Intake.
  4. 4
    Call the builder
    Confirm details, answer questions, collect deposit. Builder contact is on the job record.
  5. 5
    Update status
    Move the job:
    Intake → Active → Production → Complete
Admin Builder Accounts
  1. 1
    Go to
    localhost:3000/admin
    Enter the admin password.
  2. 2
    Create an account
    Username, temp password, name, company, email, phone. Hit Create Account.
    Hand credentials to the builder in person.
  3. 3
    Deactivate / Reactivate
    Click Deactivate to immediately block access. Click again to restore.
  4. 4
    Reset a password
    Click Reset PW, enter a new temp password, save.
Notes
  • 📁 Orders saved locally at data/orders/
  • 📧 Emails sent from residentialacc2
    @gmail.com
  • 🔒 Builder sessions last 30 days. Admin session lasts 7 days.
  • 🛠 Server runs on npm run dev — must be running for the portal to work.
